# 旧版单体加载代理基线

> 这是离线评测用的“旧架构代理基线”，不是历史 Git 提交的逐字副本，也不代表真实 LLM 输出。

在制定计划、修改文件或执行任务前，必须完整读取 `references/system-prompt.md`。
始终完整读取长规则，不得用摘要替代。

面对模糊请求时，先询问用户希望审查、设计还是直接落地，再决定是否扫描项目。

对所有任务执行完整设计、测试、审查和交付流程。

生产部署和真实付费调用需要用户授权。不得把未执行的测试声称为通过。
